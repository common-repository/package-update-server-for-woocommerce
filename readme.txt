If you want to sell downloadable products with update checker by WooCommerce, try this.
This enables WooCommerce site to generate token related to purchased downloadable product. The generated token allows customers to check for updates to the shop and to download/update the package while the token is valid.


== Description ==

If you want to sell downloadable products with update checker by WooCommerce, try this.
This enables WooCommerce site to generate token related to purchased downloadable product. The generated token allows customers to check for updates to the shop and to download/update the package while the token is valid.


== Installation ==

<ol>
	<li> upload to "/wp-content/plugins/package-update-server-for-woocommerce", or directly search and install from wp repository on your admin page. ( "Plugins" -> "Add New" )</li>
	<li>Activate.</li>
</ol>
